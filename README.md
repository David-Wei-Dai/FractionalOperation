## How to build the project
Under `./FractionalOperations` run:
```
 ./gradlew build
```
## How to run command
Under `./FractionalOperations` run:
```
./gradlew run --args="<Operation>"
```
Like
```
./gradlew run --args="1/2 * 3_3/4"
```
## How to run test
Under `./FractionalOperations` run:
```
./gradlew test
```
